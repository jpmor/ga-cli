# § 24-9-904. Definitions

(1) "Public office" shall have the same meaning as set forth in Code Section 24-8-801.
(2) "Public officer" means any person appointed or elected to be the head of any entity included in paragraph (1) of Code Section 24-9-902.
(3) "Telephone service provider" shall have the same meaning as "voice service provider" as set forth in Code Section 46-5-231.

## History

Added by 2011 Ga. Laws 52,§ 2, eff. 1/1/2013.
