# § 24-11-1. Definitions

(1) "Custodian" means the person charged with the duty of maintaining public records.
(2) "Duplicate" means a counterpart which accurately reproduces the original.
(3) "Public record" shall have the same meaning as set forth in Code Section 24-8-801.

## History

Added by 2011 Ga. Laws 52,§ 2, eff. 1/1/2013.
