# § 36-61-3.1. "Public use" defined; eminent domain to be exercised solely for public use

(a) As used in this Code section, the term "public use" shall have the meaning specified in Code Section 22-1-1.
(b) Any exercise of the power of eminent domain under this chapter must:
  (1) Be for a public use; and
  (2) Be approved by resolution of the governing body of the municipality or county in conformity with the procedures specified in Code Section 22-1-10.

## History

Added by 2006 Ga. Laws 444,§ 22, eff. 4/4/2006.
