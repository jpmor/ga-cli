# § 36-44-18. Application of Urban Redevelopment Law

It is specifically provided that Code Section 36-61-16 of the "Urban Redevelopment Law," which Code section provides for cooperation among public bodies for redevelopment purposes under said law, shall be applicable to the exercise of redevelopment powers provided by this chapter.

## History

Added by 2009 Ga. Laws 52,§ 2, eff. 4/22/2009.
