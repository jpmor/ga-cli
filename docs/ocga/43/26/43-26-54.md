# § 43-26-54. Court order; citation for civil contempt

The board may seek an order from a court of competent jurisdiction for a report from a nurse as required by Code Section 43-26-51 if one is not forthcoming voluntarily. The board may seek a citation for civil contempt if a court order for a report is not obeyed by such nurse.
