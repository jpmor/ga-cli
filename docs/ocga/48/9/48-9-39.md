# § 48-9-39. Violation of Code Section 48-9-38; penalty

(a) It shall be unlawful for any person to operate or cause to be operated on any highway in this state any motor vehicle that does not carry the registration card required by Code Section 48-9-38 or any motor vehicle that does not display in the manner prescribed by the commissioner the identification marker that Code Section 48-9-38 requires to be displayed.
(b) Any person who violates subsection (a) of this Code section shall be guilty of a misdemeanor and, upon conviction thereof, shall be punished by a fine of not less than $50.00 nor more than $200.00. Each day's operation in violation of this Code section shall constitute a separate offense.
