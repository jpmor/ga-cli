# § 50-3-93. State crustacean

White shrimp is designated as the official Georgia state crustacean.

## History

Added by 2024 Ga. Laws 495,§ 2, eff. 7/1/2024.
See 2024 Ga. Laws 495, § 1.
Pursuant to Code Section 28-9-5, in 2024, Code Section 50-3-92, as enacted by § Ga. L. 2024, p. 445, 2/HB 1341, was redesignated as Code Section 50-3-93.
