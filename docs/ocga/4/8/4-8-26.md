# § 4-8-26. Euthanasia for dogs with repeat offenses

A dog that is found, after notice and opportunity for hearing as provided by Code Section 4-8-23, to have caused a serious injury to a human on more than one occasion shall be euthanized; provided, however, that no injury occurring before July 1, 2012, shall count for purposes of this Code section.

## History

Added by 2012 Ga. Laws 765,§ 4, eff. 7/1/2012.
