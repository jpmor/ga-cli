# § 40-14-20. Definitions

(1) "Governing authority" means any county, municipality, or consolidated government.
(2) "Recorded images" has the meaning provided in subparagraph (f)(1)(B) of Code Section 40-6-20.
(3) "Traffic-control signal monitoring device" has the meaning provided in subparagraph (f)(1)(C) of Code Section 40-6-20.
