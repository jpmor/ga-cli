# § 33-65-9. Severability

If any provision of this chapter other than Code Section 33-65-6, or the application thereof to any person or circumstance, is held invalid, such determination shall not affect the provisions or applications of this chapter which can be given effect without the invalid provision or application, and to that end the provisions of this chapter, with the exception of Code Section 33-65-6, are severable.

## History

Added by 2019 Ga. Laws 315,§ 1, eff. 1/1/2020.
