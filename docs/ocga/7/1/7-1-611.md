# § 7-1-611. Penalties for violations

Any bank, bank holding company, or company (as defined in Code Section 7-1-605) which violates subsection (a) of Code Section 7-1-601 or any provision of Code Section 7-1-602, Code Section 7-1-603, or Code Sections 7-1-605 through 7-1-608 shall be subject to civil monetary penalties of not less than $500.00 nor more than $5,000.00 unless it shall cease and desist therefrom within 60 days after notice of any such violation served on it by the department. Each day on which such violation occurs shall constitute a separate offense.

## History

Amended by 2016 Ga. Laws 450,§ 7-14, eff. 7/1/2016.
