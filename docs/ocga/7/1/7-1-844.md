# § 7-1-844. Other misdemeanors of directors, officers, agents, and employees of financial institutions

(1) Code Section 7-1-37, relating to restrictions on department officials and employees;
(2) Code Section 7-1-62, relating to accounting requirements to be followed by financial institutions;
(3) Code Section 7-1-285, relating to loan limits on banks;
(4) Code Section 7-1-291, relating to borrowing limits on banks;
(5) Code Section 7-1-311, relating to operations as a fiduciary by a trust company;
(6) Code Section 7-1-491, relating to limitations on financing to directors and officers of banks and trust companies;
(7) Code Section 7-1-492, relating to prohibitions applicable to directors, officers, employees, and attorneys of bank and trust companies in dealings with the institutions with which they are connected; or
(8) Code Section 7-1-658, relating to loan limits on credit unions.
