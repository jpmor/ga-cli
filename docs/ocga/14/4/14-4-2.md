# § 14-4-2. Existing venue statutes unaffected by chapter

Nothing in this chapter shall affect existing statutes with respect to the venue of actions against railroad, electric, banking, trust, insurance, canal, navigation, express, and telegraph companies, which existing statutes include, as to express companies, those statutes codified as Code Sections 46-9-234 through 46-9-236; as to companies under the jurisdiction of the Georgia Public Service Commission, that statute codified as Code Section 46-2-92.

## History

Amended by 2012 Ga. Laws 700,§ 1, eff. 7/1/2012.
