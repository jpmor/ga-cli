# § 14-3-864. Effect of court approval of transaction

In a case involving a charitable corporation, a transaction that was not the subject of either directors' action under Code Section 14-3-862 or members' action under Code Section 14-3-863 is effective for purposes of paragraph (3) of subsection (b) of Code Section 14-3-861 if the transaction is approved by the superior court, in an action in which the Attorney General is joined as a party.
