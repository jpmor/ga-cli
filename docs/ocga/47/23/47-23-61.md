# § 47-23-61. Establishing creditable service

Any member who on June 30, 1998, was entitled to receive creditable service for prior service as a state court judge, state court solicitor-general, or juvenile court judge pursuant to the provisions of Code Section 47-9-41.1, as such Code section appeared on June 30, 1998, shall be entitled to establish creditable service under this retirement system, under the same conditions, for such prior service rendered prior to July 1, 1998. This Code section shall not apply to any other member of this retirement system.
